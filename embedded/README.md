
## Setup DevEnv

- Install: `vscode-arduino.vscode-arduino-community`
- Install board: `Arduino: Board Manager` -> `Arduino SAMD Boards (32-bits ARM Cortex-M0+)`
- Install libraries: `Arduino: Library Manager`:
  - RTCZero
  - WiFiNINA
  - FlashStorage
  - LiquidCrystal
- Implement: `src/view/ConnectInfo.cpp`
